# Workshop Exercises

This folder contains guided exercises for the **GitHub Copilot: Context Mastery & Beyond Developers** workshop.

## Exercise Map

| Exercise | Block | Focus | Time |
|----------|-------|-------|------|
| [Exercise 1](exercise-1.md) | Block 1 | Your First Non-Code Task | 8 min |
| [Exercise 2](exercise-2.md) | Block 2 | See the Difference Context Makes | 15 min |
| [Exercise 3](exercise-3.md) | Block 3 | Create Instructions Files | 10 min |
| [Exercise 4](exercise-4.md) | Block 3 | Create Role-Specific Prompt Files | 10 min |
| [Exercise 5](exercise-5.md) | Block 4 | Agent & Skill Exploration | 8 min |
| [Capstone](capstone.md) | Block 5 | Putting It All Together | 15 min |

## How to Use These Exercises

1. Each exercise has a **Goal**, **Steps**, and **Role-Specific Suggestions**
2. Pick the path that matches your role (Developer, PM, Tech Writer, DBA, or Manager)
3. Don't worry about getting a "perfect" answer — the goal is to learn how context affects output
4. Share your results with the group during debrief sections

## Folder Structure for Your Work

As you complete exercises, you'll create files in these locations:

```
.github/
├── copilot-instructions.md          ← Exercise 3 (repo-level instructions)
└── prompts/
    └── your-prompt.prompt.md        ← Exercise 4 (prompt files)
.vscode/
    └── your-mode.chatmode.md        ← Bonus (custom chat modes)
docs/.instructions.md                ← Exercise 3 (scoped instructions)
sql/.instructions.md                 ← Exercise 3 (scoped instructions)
reports/.instructions.md             ← Exercise 3 (scoped instructions)
```
